CleverCogs

Data Analysis, User Interviews

This project is an analysis of the usage data of a tablet OS for elderly and people with disabilities. I worked as part of a group to help the product owner gain insight into the users and usage of the OS. We achieved this by conducting user interviews and analysing the usage data.
